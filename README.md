# python-practice
A repo for practicing major Python concepts and solving common Python problems.

- big_o_notation.py focuses on algorithmns that showcase different time complexities.
- code_wars_solutions.py contain solutions to sample problems that can be found on codewars.com (Similar to Leetcode)
- concepts_and_terms.py explores various programming concepts such as closures, memoization, pythonic idioms, async, multiprocessing, threading, accumulators, iterators, and more!
- oop_refresh.py steps through the 4 pillers of OOP in a pythonic way. Abstration, Encapsulation, Inheritance, Polymorphism
- selenium_tricks.py Explores fancy things you can do in selenium with xpaths and other tricks.
